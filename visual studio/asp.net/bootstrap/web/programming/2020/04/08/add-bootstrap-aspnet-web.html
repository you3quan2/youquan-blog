How To Add Bootstrap In ASP.NET

A quick guide on how to add Bootstrap in ASP.NET.

Apr 8, 2020 • You Quan • 2 min read

visual studio   asp.net   Bootstrap   web   programming

If you have learn basic web design, then you must know about Cascading Style Sheets (CSS). Commonly, your web project will have a .css file, which defines the design of your web pages. For example, if we want to have a button with green background like this: , we need have a .css file which defined the style of the button:

.button {
  background-color: #4CAF50;
  border: none;
  color: white;
  padding: 15px 32px;
  text-align: center;
  text-decoration: none;
  display: inline-block;
  font-size: 16px;
  margin: 4px 2px;
  cursor: pointer;
}

<button type="button" class="btn btn-success">Success</button>

Bootstrap is a free and open-source CSS framework dire

Follow the steps below to add Bootstrap to our ASP.NET web project:

  1. In the Solution Explorer, right click on the project (e.g. ASPNetTutorial) and select **Manage NuGet Packages.

  2. Under the NuGet window, click on Browse and search for Bootstrap using the Search. Once we get the Bootstrap package , click on ↓ to add the package ro our web project.

  3. We will be asked to confirm installation, where the following files will be added into our web project:
    • jQuery.3.0.0
    • popper.js.1.16.0
    • bootstrap.4.4.1

  4. If the installation is successful, we will see multiple files (.css, map, .js) aaded to the Content and Scripts folder under our web project.

  5. Drag and drop these three specific files from the Content and Scripts into the <head></head> section of the web page where we want to code with Bootstrap (I have added it to the Login.apsx which we have created in the very first tutorial.
    • bootstrap.min.css
    • jquery-3.0.0.min.js
    • bootstrap.min.js
     <html>
 <head runat="server">
     <title></title>

     <link href="Content/bootstrap.min.css" rel="stylesheet" />
     <script src="Scripts/jquery-3.0.0.min.js"></script>
     <script src="Scripts/bootstrap.min.js"></script>
    
 </head>
 <body>
     ...other codes...
 </body>
 </html>

  6. To check if we have added the Bootstrap correctly, run the web project and we should be able to spot the differences in the Login page:

  7. If the differences (e.g. font and size) are not obvious for you, let’s add a Boostrap class to the Login button:

     <asp:Button ID="login_button" class="btn btn-outline-danger" runat="server" Text="LOGIN" />

    Now, run the project again and hopefully you can notice how the design of the Login button had changed:
